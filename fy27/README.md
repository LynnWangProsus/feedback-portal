# FY27 Feedback Portal

Anonymous feedback collection for FY27. Same portal experience as the original (Feedback From / Feedback To, team categories, strengths/improvements/comments), but writes to its own `feedback_fy27` Firestore collection — fully separate from FY26 data, no overlap.

Once hosted, this will live at a sub-path of the existing site, e.g. `https://lynnwangprosus.github.io/feedback-portal/fy27/`.

Admin password is currently the same as the FY26 portal (`ilovevals!`, set in `index.html`) — change it there if you want FY27 access locked down separately.
